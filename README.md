# Система распознавания действий человека на основе анализа скелетных данных

## Общее описание проекта

### О проекте

Skeleton Action Recognition — это программная система для распознавания действий человека в реальном времени на основе анализа скелетных данных, извлекаемых из видеопотока. Она включает в себя:

- Извлечение скелетных ключевых точек из видео с использованием MediaPipe Pose
- Предобработка данных (нормализация, интерполяция, фильтрация, слайсинг)
- Аугментация данных для увеличения обучающей выборки
- Четыре архитектуры нейронных сетей для классификации: GRU, CNN+LSTM, TCN, ST-GCN
- Обучение и валидация моделей
- Инференс в реальном времени с веб-камеры
- Визуализация результатов (матрицы ошибок, графики обучения, сравнение моделей)

### Структура датасета

|---------|----------|

| Разрешение видео | 1280×720 |
| Разделение данных | Train 60% / Val 20% / Test 20% |

### Классы действий

| № | Класс | Описание |
|---|-------|----------|
| 1 | стоять | Статическая поза, руки опущены |
| 2 | сидеть | Поза, сидя прямо на стуле |
| 3 | лежать | Горизонтальное положение |
| 4 | идти | Ходьба в пределах кадра |
| 5 | руки за головой | Руки поднимаются и заводятся за голову |
| 6 | поднять руки | Обе руки поднимаются вертикально вверх |
| 7 | похлопать | Хлопающие движения руками |
| 8 | наклон вперёд | Наклон корпуса вперёд |
| 9 | удар кулаком | Удар кулаком вперёд |
| 10 | удар ногой | Удар ногой вперёд |

## Быстрый старт

### Минимальные требования

- Python = 3.10
- RAM: минимум 8 GB (рекомендуется 16 GB)
- CPU: 4 ядра
- GPU: опционально (CUDA для ускорения обучения)

### Установка зависимостей

```bash
# 1. Клонирование репозитория
git clone https://github.com/dimarik1117/human-action-recognition-system.git
cd human-action-recognition-system

# 2. Создание виртуального окружения
python -m venv venv

# Активация на Windows
venv\Scripts\activate

# Активация на Linux/Mac
source venv/bin/activate

# 3. Установка зависимостей
pip install -r requirements.txt
```

## Использование

### Запуск инференса в реальном времени

```bash
# Запуск с веб-камеры (модель ST-GCN по умолчанию)
python inference_real_time.py

# Запуск с моделью GRU
python inference_real_time.py --model gru

# Запуск с сохранением видео
python inference_real_time.py --model cnn_lstm --save_video

# Пользовательские параметры
python inference_real_time.py --model tcn --camera 0 --window_size 30 --smoothing 5
```

### Управление в режиме реального времени

| Клавиша | Действие |
|---------|----------|
| q / ESC | Выход из программы |
| s | Сохранить текущий кадр (screenshots/) |
| m | Вкл/Выкл отображение скелета |
| c | Вкл/Выкл отображение соединений суставов |

### Запуск обучения

Поместите видеофайлы .mp4 в директорию согласно её классу, соблюдая стиль названий (class_01_stand/subject_01_001.mp4)

```bash
# Полный пайплайн (извлечение скелета + предобработка + обучение + оценка)
python main.py

# Только обучение GRU
python main.py --models gru

# Обучение двух выбранных моделей
python main.py --models gru stgcn

# С отключённой аугментацией
python main.py --no_augmentation

# Пользовательские параметры
python main.py --epochs 100 --batch_size 32 --lr 0.001
```

Сравнения всех моделей (графики обучения, таблицы и диаграммы результатов, матрицы ошибок) будут находиться в папке results.
